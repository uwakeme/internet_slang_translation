# 互联网黑话翻译器

一个功能强大的微信小程序，专门帮助用户理解和使用互联网行业的专业术语，支持黑话与人话的智能双向翻译。

## ✨ 项目亮点

🎯 **智能翻译** - 自动识别文本类型，一键完成双向翻译  
📚 **丰富词库** - 内置72+个精选互联网黑话，涵盖6大核心领域  
🔄 **双向转换** - 支持黑话→人话、人话→黑话两种翻译模式  
💾 **历史记录** - 自动保存翻译历史，方便回顾学习  
🎨 **现代UI** - 渐变设计风格，流畅的用户体验  

## 🚀 功能特性

### 🏠 首页 - 快速翻译
- 智能识别输入内容类型
- 一键快速翻译
- 支持复制翻译结果
- 便捷的功能入口导航

### 📚 分类浏览
- **会议用语** - 开会时常用的黑话 (15个词汇)
- **项目管理** - 项目相关的专业术语 (12个词汇)
- **产品运营** - 产品和运营的行话 (18个词汇)
- **技术开发** - 程序员的专业黑话 (10个词汇)
- **商务合作** - 商务谈判常用词汇 (8个词汇)
- **人力资源** - HR领域的专业术语 (9个词汇)

### 🔄 专业翻译
- **黑话→人话** - 将互联网术语翻译成通俗表达
- **人话→黑话** - 将日常用语转换为专业术语
- 翻译历史记录 (最多保存20条)
- 支持从分类页面快速跳转翻译

## 🛠 快速开始

### 环境要求
- 微信开发者工具 (最新版本)
- 微信小程序开发账号
- Node.js (可选，用于扩展开发)

### 安装步骤
1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/internet-jargon-translator.git
   cd internet-jargon-translator
   ```

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录

3. **配置项目**
   - 修改 `project.config.json` 中的 `appid`
   - 确保 `images/` 目录包含所需图标

4. **运行项目**
   - 点击"编译"按钮
   - 在模拟器中预览效果

### 自定义配置
- **扩展词汇**: 在 `utils/data.js` 中添加新的分类和词汇
- **修改样式**: 在各页面的 `.wxss` 文件中调整UI样式
- **添加功能**: 在对应页面的 `.js` 文件中扩展业务逻辑
