# 互联网黑话翻译器

一个微信小程序，帮助用户理解和使用互联网行业的专业术语，支持黑话与人话的双向翻译。

## 项目简介

互联网黑话翻译器是一个实用的微信小程序，旨在帮助用户：
- 理解互联网行业的专业术语和"黑话"
- 将复杂的行业术语翻译成通俗易懂的表达
- 将日常用语转换为专业的互联网术语
- 学习和掌握互联网行业的沟通方式

## 功能特性

### 🏠 首页 - 快速翻译
- 智能识别输入内容类型
- 一键快速翻译
- 支持复制翻译结果
- 便捷的功能入口导航

### 📚 分类浏览
- **会议用语** - 开会时常用的黑话 (15个词汇)
- **项目管理** - 项目相关的专业术语 (12个词汇)
- **产品运营** - 产品和运营的行话 (18个词汇)
- **技术开发** - 程序员的专业黑话 (10个词汇)
- **商务合作** - 商务谈判常用词汇 (8个词汇)
- **人力资源** - HR领域的专业术语 (9个词汇)

### 🔄 专业翻译
- **黑话→人话** - 将互联网术语翻译成通俗表达
- **人话→黑话** - 将日常用语转换为专业术语
- 翻译历史记录 (最多保存20条)
- 支持从分类页面快速跳转翻译

## 核心模块说明

### 📱 页面模块 (pages/)

#### 首页 (index/)
- **功能**: 快速翻译入口，智能识别文本类型
- **特性**: 自动判断输入是黑话还是人话，一键翻译
- **交互**: 支持复制结果，快速导航到其他功能

#### 分类页 (category/)
- **功能**: 按类别展示互联网黑话词汇
- **特性**: 6大分类，72个精选词汇
- **交互**: 点击词汇可跳转到翻译页面进行详细翻译

#### 翻译页 (translate/)
- **功能**: 专业的双向翻译工具
- **特性**: 支持两种翻译模式，保存翻译历史
- **交互**: 模式切换，历史记录管理，结果复制

### 🛠 工具模块 (utils/)

#### 数据管理 (data.js)
```javascript
// 主要功能
- getCategories()        // 获取所有分类
- getWordsByCategory()   // 根据分类ID获取词汇
- getAllWords()          // 获取所有词汇
```

#### 翻译引擎 (translator.js)
```javascript
// 核心方法
- translateToHuman()     // 黑话翻译为人话
- translateToJargon()    // 人话翻译为黑话
- translateText()        // 智能翻译（自动识别类型）
- detectTextType()       // 检测文本类型
```

## 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序开发账号

### 安装步骤
1. 克隆或下载项目代码
2. 在微信开发者工具中导入项目
3. 修改 `project.config.json` 中的 `appid`
4. 添加底部导航栏图标到 `images/` 目录
5. 编译运行

### 自定义配置
- **扩展词汇**: 在 `utils/data.js` 中添加新的分类和词汇
- **修改样式**: 在各页面的 `.wxss` 文件中调整样式
- **添加功能**: 在对应页面的 `.js` 文件中扩展功能
